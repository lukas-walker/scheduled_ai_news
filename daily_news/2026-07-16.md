# AI News — 2026-07-16

## Software Engineering
- **DeepSeek open-sources DSpark speculative decoding** — DeepSeek and Peking University released "DSpark," a speculative-decoding technique that speeds up generation by more than 50% without accuracy loss, shipped as DeepSeek-V4-Pro/Flash-DSpark checkpoints. [Source](https://www.deeplearning.ai/the-batch/issue-361)

## Ethics
- **Grok Build CLI caught silently exfiltrating user files** — Simon Willison reports that xAI's Grok Build coding-agent CLI was uploading entire local directories — including SSH keys and password-manager databases — to xAI's Google Cloud buckets without clear disclosure. xAI disabled default data retention starting July 12, is deleting previously retained data, and open-sourced the tool under Apache 2.0, though Willison notes the (now-disabled) upload code is still present in the source. [Source](https://simonwillison.net/2026/Jul/15/grok-build/)

## Geopolitics & Security Policy
- **Claude Fable 5 and Claude Mythos restored after export-control suspension** — Per DeepLearning.AI's The Batch, both models came back online July 1 after a three-week suspension tied to a US Commerce Department export-control directive; Fable 5 currently leads coding benchmarks at 80.3% on SWE-Bench Pro. [Source](https://www.deeplearning.ai/the-batch/issue-361)
